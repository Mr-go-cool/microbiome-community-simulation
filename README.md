# Microbiome Community Simulation Pipeline

## Overview
This project focuses on microbiome community metabolic modeling using MICOM and AGORA-based genome-scale metabolic reconstructions.

The workflow includes:
- Microbiome abundance preprocessing
- Taxonomic filtering and mapping
- MICOM community model preparation
- Diet-based metabolic simulations
- Probiotic and prebiotic intervention analysis
- Flux and growth rate analysis

4. Build community metabolic models
